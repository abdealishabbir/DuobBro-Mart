#  DUOBRO MART

A lightweight, zero-investment e-commerce storefront built with pure HTML, CSS, and JavaScript — no frameworks, no backend, no monthly fees.

---

# About

DUOBRO MART is a Pakistani online store selling **fashion** and **household products**. The entire site lives in a single HTML file, making it fast to load, easy to maintain, and deployable anywhere for free.

---

# Features

- **Single-file architecture** — Everything in one `index.html`
- **Product catalog** with two categories:
  -  Fashion
  -  Household (utensils, ironing stands, drying stands, and more)
- **Two-tier navigation tabs** for smooth category browsing
- **Order collection via Formspree** — orders land directly in email, no backend needed
- **WhatsApp integration** for quick customer communication
- **Mobile responsive** design with `100dvh` hero and optimized image loading
- **Cloudflare-compatible** email handling via runtime JS string assembly

# Getting Started

No installation or build step required.

1. Clone the repo:
```bash
   git clone https://github.com/abdealishabbir/DuobBro-Mart.git
```
2. Open `index.html` in any browser.

That's it.

---

# Deployment

The site can be hosted on any static hosting platform:

- **GitHub Pages** — push to a repo and enable Pages in settings
- **Cloudflare Pages** — connect the repo for auto-deploy
- **Netlify / Vercel** — drag and drop the file

---

# Order Flow

1. Customer browses products
2. Fills out the order form
3. Formspree delivers the order details to the store's email
4. Store owner follows up via WhatsApp or email
